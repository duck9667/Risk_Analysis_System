# Risk_Analysis_System
웹기반 악취 민원 분석 시스템

# 프로젝트 배경
중앙대학교 환경시스템 연구실과 협업하여 진행한 프로젝트로 서울시 악취 민원의 효율적인 분석을 위해 웹기반 대시보드 구축하고자 함

# 개발 환경
구름 IDE(goorm IDE) 클라우드

# 주요 기능
- 시각화
- 이미지 저장

# 패키지
- flask : 마이크로 웹 프레임워크
- plotly : 데이터 시각화 패키지로 Bokeh 처럼 Java scrpit 기반으로 작동한다.
